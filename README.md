# 고양이를 만지고 잡고 놀아보자
## - Pet Your Cat -

털 날리고 까칠한 고양이를 앱 또는 웹을 통해 만나며<br>
방을 변경하고 만지고 옮지고 하여 고양이의 움직임을 살펴보고<br>
컨디션을 확인하고 케어하는 집사의 삶을 살아보자

# 구현하고 싶은 기능
## 고양이 관련 기능
 - 고양이가 스스로 움직이는 기능
 - 고양이를 잡고 옮기는 기능
 - 고양이를 쓰다듬는 기능
 - 고양이의 밥과 화장실을 챙겨주는 기능
 - 고양이의 종 또는 모습을 설정하는 기능

## 방 관련 기능
 - 방 사진을 직접 설정하고 변경하는 기능 (최대 3개까지)
